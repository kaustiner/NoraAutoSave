🚀 INSTALAÇÃO COMPLETA DA NORA (SIMPLIFICADO)
Para instalar a NORA no seu computador, siga estes passos:
Primeiro, certifique-se de que você tem o Python instalado (versão 3.10 ou superior). Depois, abra o terminal (CMD ou PowerShell) e execute:
Bash
git clone https://github.com/kaustiner/NoraAutoSave
Em seguida, entre na pasta do projeto:
Bash
cd NoraAutoSave
Agora você pode instalar tudo automaticamente de duas formas:
✔️ Forma mais fácil (recomendado no Windows):
Clique ou execute o arquivo:
Plain text
install.bat
✔️ Forma manual (caso prefira terminal):
Bash
pip install -r requirements.txt
Depois da instalação, para iniciar a NORA, execute:
Bash
python main.py
🧠 COMO A NORA FUNCIONA
Depois de iniciar, a NORA vai ficar em modo terminal esperando comandos.
Você pode digitar coisas como:
abrir youtube
youtube abrir
abre o youtube
nora abre youtube
Ela vai analisar o comando, encontrar o plugin correspondente e executar automaticamente.
